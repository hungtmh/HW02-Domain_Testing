# BUG-008: Màu nút Đăng ký trên giao diện không nhất quán với thiết kế chung

## Metadata
| Field | Value |
|-------|-------|
| Feature | FR-01 / UI Mismatch |
| Severity | Trivial |
| Priority | Low |
| Component | Web |
| Test Case | DT-11 |
| Status | Open |

## SRS Reference
> Phù hợp với hướng dẫn thiết kế giao diện (nút submit Đăng ký nên dùng màu xanh primary của hệ thống).

## Environment
- OS: Windows 10
- Browser: Chrome
- Frontend Web: localhost:5173/register

## Steps to Reproduce
1. Truy cập trang Đăng ký `http://localhost:5173/register`.
2. Kiểm tra trực quan màu sắc của nút "Đăng Ký".

## Expected Result (SRS)
Nút Đăng ký có màu xanh (blue) chủ đạo giống với màu primary của hệ thống.

## Actual Result
Nút Đăng ký có màu đỏ (`bg-red-500` / hover: `bg-red-600`).

## Evidence
- Mã nguồn tại [Register.jsx dòng 70-75](file:///d:/Kiem_thu/HW2/HW02-Group08/frontend-web/src/pages/Register.jsx#L70-L75):
```javascript
<button
  type="submit"
  className="w-full bg-red-500 text-white py-2 rounded hover:bg-red-600"
>
  Đăng Ký
</button>
```

## Notes
- Đây là lỗi thẩm mỹ giao diện (Trivial). Cần chuyển sang lớp màu xanh dương (như `bg-blue-500` / hover: `bg-blue-600`) để đồng bộ với giao diện chung của ứng dụng.
